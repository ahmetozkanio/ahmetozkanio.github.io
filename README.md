# Portfolio

A retro-futuristic portfolio website built with HTML, CSS, and JavaScript.

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Edit `data.json` to update content

## Project Structure

- `index.html` - Main HTML file
- `js/main.js` - JavaScript for dynamic content
- `data.json` - All website content
- `flutter_build/` - Archived Flutter build

## Updating Content

Edit `data.json` to change:
- Profile information
- Tech stack
- Projects
- Experience
- Education

## Deployment

Ready for GitHub Pages - just push to main branch.
